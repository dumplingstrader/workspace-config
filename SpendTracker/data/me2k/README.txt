Place ME2K export files here.

File naming:
  - Filename must start with "EXPORT_"
  - Must be .xlsx format

Examples:
  - EXPORT_20260126.xlsx
  - EXPORT_20260126_AllOrders.xlsx

The script will automatically use the most recently modified file.

How to export from SAP:
  1. Run transaction ME2K
  2. Enter Internal Order and Plant (R180)
  3. Execute (F8)
  4. List > Export > Spreadsheet
  5. Save as Excel (.xlsx)
  6. Move file to this folder
