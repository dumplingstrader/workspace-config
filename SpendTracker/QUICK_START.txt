LA REFINERY SPEND TRACKER v2.0 - QUICK START
==================================================

1. SETUP (first time only)
   - Ensure Python 3.8+ is installed (https://www.python.org/downloads/)
   - Right-click setup.ps1 > "Run with PowerShell"
   - Or run in PowerShell: .\setup.ps1

2. PREPARE DATA
   - Export ME2K from SAP, save to data\me2k\ folder
     (filename must start with "EXPORT_")
   - Get TS Actuals from supervisor, save to data\ts_actuals\ folder
     (filename must contain "ts_actual")

3. RUN
   - Right-click run_tracker.ps1 > "Run with PowerShell"
   - Or run in PowerShell: .\run_tracker.ps1

4. OUTPUT
   Excel file generated in output\ folder with tabs:
   - Summary: High-level by Internal Order
   - ME2K_Commitments: PO detail
   - TS_Actuals: GL postings
   - Cross_Charges: Non-PO entries (REVIEW THIS!)
   - Monthly_Pivot: Spend by month
   - Watson_Cogen: IO# 9913117 tracking
   - IO_Reference: Order number lookup

5. INTERNAL ORDERS TRACKED

   9909813 - PC - Outside Support (Contractors)
   9909815 - PC - Hardware & Materials
   9909818 - PC - APC Work & Maint
   9912360 - PC - Dept Misc Exp & Travel
   9913117 - Watson Cogen
