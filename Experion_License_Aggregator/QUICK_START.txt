EXPERION LICENSE AGGREGATOR v2.0 - QUICK START
====================================================

1. SETUP
   - Copy your Experion license XML files into:
     data\raw\Carson\     (organized by system folder)
     data\raw\Wilmington\ (organized by system folder)
   - Edit config\system_names.json to add friendly names for your systems
   - Edit config\cost_catalog.json with your actual Honeywell pricing

2. RUN
   Double-click experion-license-tool.exe

   Or from command line:
   experion-license-tool.exe              # Standard run
   experion-license-tool.exe --diff       # Show changes since last run
   experion-license-tool.exe --verbose    # Detailed output
   experion-license-tool.exe --help       # Show all options

3. OUTPUT
   Excel file generated in data\output\ with sheets:
   - Executive Summary: Key metrics and Top 5 action items
   - PKS/HS/EAS: License comparison with conditional formatting
   - Summary: Totals by cluster with cost estimates
   - Transfer Candidates: Systems with excess capacity
   - Changes: Differences from previous run (if any)

4. COLOR LEGEND
   - Green:  Transfer candidate (excess points)
   - Red:    Stale license (>2 years old)
   - Orange: Invalid customer name
   - Yellow: Changed since last run

5. DATABASE
   license_history.db stores 3 years of snapshots for change tracking.
   Located in same folder as the EXE.
